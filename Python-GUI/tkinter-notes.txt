-> TKinter == TK Interface
-> TTK -> Can be styled

-> TKinter Pack:
    -> Arranges application elements
    -> Supports padding
    
    -> Expansion Priority
        -> If elements have "expand=true"
        -> If elements are set to expand...
        -> These elements will have expansion Priority

-> Padding for an Element:
    -> Left
    -> Top
    -> Right
    -> Bottom

-> TKinter Geometry Grid
    -> uses "attribute.grid()" syntax
    -> Column default weight is 0
    -> Column Weight: "root.columnconfigure(0,weight=1)"

-> Activate high DPI
    -> Wrap in try/catch
    -> "from ctypes import windll"
    -> "windll.shcore.SetProcessDpiAwareness(1)"
